# 📌 Project Overview

This project demonstrates the design and development of an interactive Power BI dashboard that enables stakeholders to monitor sales performance, profitability, product performance, and regional trends.

The dashboard allows users to filter data dynamically and gain insights through interactive visualizations.

---

# 🚀 Features

- Executive KPI Dashboard
- Interactive Slicers
- Monthly Sales & Profit Trend
- Country-wise Sales Analysis
- Product Profitability Analysis
- Segment-wise Performance
- Profit Margin Analysis
- Dynamic Cross Filtering
